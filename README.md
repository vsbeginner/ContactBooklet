# ContactBooklet
Python-based Contact Booklet Assignment for MCA (AI &amp; ML)
A simple, robust Command Line Interface (CLI) application written in Python to manage your contacts efficiently. This project demonstrates file handling (CSV & JSON), error logging, and standard CRUD (Create, Read, Update, Delete) operations.
Features

Add Contacts: Save names, phone numbers, and emails.

View All: Display contacts in a clean, formatted table.

Search: Find contacts by name instantly.

Update: Edit existing details (Name, Phone, or Email) using the contact's Index Number.

Delete: Remove unwanted contacts safely.

Auto-Sync: Automatically saves data to both contacts.csv and contacts.json upon any change.

Import/Export: Manually export data to JSON or import from an external JSON file.

Error Logging: Tracks program errors in error_log.txt with timestamps.
